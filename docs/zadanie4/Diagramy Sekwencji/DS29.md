sequenceDiagram
    actor K as Kandydat na Moderatora
    participant UI as Formularz Aplikacyjny
    participant S as System
    participant FS as Serwis Plików
    participant DB as Baza Danych

    K->>UI: Wpisuje dane osobowe
    K->>UI: Wpisuje uzasadnienie aplikacji
    K->>UI: Wybiera pliki do załączenia

    UI->>S: Przesyła pliki
    S->>FS: Sprawdź typ i rozmiar plików
    
    alt Pliki niepoprawne
        FS-->>S: Błąd (typ/rozmiar)
        S-->>UI: Komunikat o błędzie pliku
        UI-->>K: Wyświetla informację o odrzuceniu pliku
        K->>UI: Wybiera poprawne pliki
        UI->>S: Przesyła ponownie
        S->>FS: Ponowna walidacja
    end

    FS->>FS: Zapisuje pliki w storage
    FS-->>S: Pliki zapisane

    S->>S: Waliduje kompletność danych formularza

    alt Dane niekompletne
        S-->>UI: Komunikat o brakujących polach
        UI-->>K: Podświetla brakujące pola
        K->>UI: Uzupełnia brakujące dane
        UI->>S: Przesyła uzupełnione dane
        S->>S: Ponowna walidacja
    end

    S->>DB: Zapisuje formularz z załącznikami
    DB-->>S: Potwierdzenie zapisu
    S-->>UI: Formularz zapisany
    UI-->>K: Formularz gotowy - przejście do testu
