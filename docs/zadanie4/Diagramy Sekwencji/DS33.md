sequenceDiagram
    actor M as Moderator
    participant UI as Panel Weryfikacji
    participant S as System
    participant DB as Baza Danych

    M->>UI: Wybiera jedną z trzech decyzji
    UI->>S: Przesyła decyzję moderatora

    alt Zatwierdzenie
        S->>DB: Ustawia status materiału: zatwierdzony
        DB-->>S: Potwierdzenie
        S->>DB: Publikuje materiał (widoczny dla użytkowników)
        DB-->>S: Potwierdzenie publikacji
        S-->>UI: Materiał zatwierdzony i opublikowany
        UI-->>M: Wyświetla potwierdzenie zatwierdzenia
    else Odrzucenie
        S->>DB: Ustawia status materiału: odrzucony
        DB-->>S: Potwierdzenie
        S-->>UI: Materiał odrzucony
        UI-->>M: Wyświetla potwierdzenie odrzucenia
    else Wysłanie do poprawy
        S->>DB: Ustawia status materiału: do poprawy
        DB-->>S: Potwierdzenie
        S->>DB: Dołącza uwagi moderatora
        DB-->>S: Potwierdzenie
        S-->>UI: Materiał wysłany do poprawy
        UI-->>M: Wyświetla potwierdzenie wysłania do poprawy
    end
