sequenceDiagram
    actor A as Administrator
    participant UI as Panel Zgłoszeń
    participant S as System
    participant DB as Baza Danych
    participant NS as Serwis Powiadomień

    A->>UI: Otwiera szczegóły wybranego zgłoszenia
    UI->>S: Żądanie kontekstu zgłoszenia
    S->>DB: Pobierz dane zgłaszanego materiału/użytkownika
    DB-->>S: Zwraca dane kontekstowe
    S-->>UI: Prezentuje kontekst
    UI-->>A: Wyświetla materiał/profil użytkownika w kontekście zgłoszenia

    A->>A: Analizuje treść zgłoszenia i kontekst
    A->>UI: Podejmuje decyzję

    alt Odrzucenie zgłoszenia
        A->>UI: Oznacza zgłoszenie jako bezzasadne
        UI->>S: Decyzja: odrzucone
        S->>DB: Zapisuje wynik: odrzucone
        DB-->>S: Potwierdzenie
    else Ostrzeżenie
        A->>UI: Wydaje ostrzeżenie użytkownikowi
        UI->>S: Decyzja: ostrzeżenie
        S->>DB: Zapisuje ostrzeżenie przy koncie użytkownika
        DB-->>S: Potwierdzenie
        S->>NS: Powiadom użytkownika o ostrzeżeniu
        NS-->>S: Powiadomienie wysłane
    else Blokada konta
        A->>UI: Blokuje konto zgłoszonego użytkownika
        UI->>S: Decyzja: blokada
        S->>DB: Blokuje konto użytkownika
        DB-->>S: Potwierdzenie
        S->>NS: Powiadom użytkownika o blokadzie
        NS-->>S: Powiadomienie wysłane
    else Usunięcie materiału
        A->>UI: Usuwa zgłoszony materiał
        UI->>S: Decyzja: usunięcie materiału
        S->>DB: Usuwa materiał z systemu
        DB-->>S: Potwierdzenie usunięcia
        S->>NS: Powiadom autora o usunięciu materiału
        NS-->>S: Powiadomienie wysłane
    end

    S->>DB: Zapisuje wynik rozpatrzenia zgłoszenia
    DB-->>S: Potwierdzenie
    S-->>UI: Zgłoszenie rozpatrzone
    UI-->>A: Wyświetla potwierdzenie rozpatrzenia
