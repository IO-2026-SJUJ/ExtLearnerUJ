sequenceDiagram
    actor U as Użytkownik
    participant UI as Panel Statystyk
    participant S as System
    participant RS as Serwis Raportów
    participant DB as Baza Danych

    U->>UI: Wybiera opcję pobrania raportu
    UI->>S: Żądanie wygenerowania raportu
    S->>RS: Generuj raport dla użytkownika
    RS->>DB: Pobierz pełne dane nauki użytkownika
    DB-->>RS: Zwraca dane (testy, postęp, materiały, aktywność)
    RS->>RS: Kompiluje dane w raport
    RS->>RS: Formatuje raport (PDF/CSV)
    RS->>RS: Tworzy plik do pobrania
    RS-->>S: Plik raportu gotowy
    S-->>UI: Udostępnia plik do pobrania
    UI-->>U: Wyświetla link pobierania / rozpoczyna pobieranie
    U->>UI: Zapisuje raport lokalnie
