sequenceDiagram
    actor M as Moderator
    participant UI as Kolejka Prac
    participant S as System
    participant DB as Baza Danych

    M->>UI: Otwiera kolejkę prac do sprawdzenia
    UI->>S: Żądanie listy prac oczekujących
    S->>DB: Pobierz prace ze statusem "oczekuje na sprawdzenie"
    DB-->>S: Zwraca listę prac
    S-->>UI: Prezentuje dostępne prace
    UI-->>M: Wyświetla listę prac do sprawdzenia

    M->>UI: Wybiera konkretną pracę

    alt Praca zarezerwowana przez innego moderatora
        S->>DB: Sprawdź status rezerwacji pracy
        DB-->>S: Praca zarezerwowana
        S-->>UI: Komunikat - praca już zarezerwowana
        UI-->>M: Wyświetla informację - wybierz inną pracę
    else Praca dostępna
        Note over M, DB: UC39 - Rezerwacja pracy (extend)
        M->>UI: Klika opcję rezerwacji
        UI->>S: Żądanie rezerwacji pracy
        S->>DB: Przypisz pracę do moderatora
        DB-->>S: Potwierdzenie rezerwacji
        S-->>UI: Praca zarezerwowana
        UI-->>M: Wyświetla potwierdzenie rezerwacji
    end
