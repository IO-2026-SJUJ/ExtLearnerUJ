sequenceDiagram
    actor M as Moderator
    participant UI as Panel Weryfikacji
    participant S as System
    participant DB as Baza Danych

    M->>UI: Wpisuje komentarz do materiału
    M->>UI: Zatwierdza komentarz
    UI->>S: Przesyła komentarz
    S->>S: Waliduje treść komentarza
    S->>DB: Zapisuje komentarz (powiązany z materiałem i moderatorem)
    DB-->>S: Potwierdzenie zapisu komentarza
    S-->>UI: Komentarz zapisany
    UI-->>M: Wyświetla potwierdzenie

    Note over M, DB: Autor odczyta komentarz po zakończeniu weryfikacji
