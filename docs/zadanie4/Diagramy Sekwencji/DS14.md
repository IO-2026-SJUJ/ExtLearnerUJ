sequenceDiagram
    actor U as Użytkownik
    participant UI as Panel Statystyk
    participant S as System
    participant SS as Serwis Statystyk
    participant DB as Baza Danych

    U->>UI: Otwiera panel statystyk

    alt Użytkownik nie jest zalogowany
        UI-->>U: Przekierowanie do UC03 (Logowanie)
    else Użytkownik zalogowany
        UI->>S: Żądanie statystyk użytkownika
        S->>SS: Pobierz statystyki
        SS->>DB: Pobierz postęp nauki
        DB-->>SS: Zwraca dane postępu
        SS->>DB: Pobierz wyniki testów
        DB-->>SS: Zwraca wyniki testów
        SS->>DB: Pobierz dane aktywności
        DB-->>SS: Zwraca dane aktywności
        SS-->>S: Kompletne statystyki
        S-->>UI: Prezentuje statystyki
        UI-->>U: Wyświetla postęp nauki, wyniki testów, aktywność

        U->>U: Analizuje swoje dane

        opt UC15 - Pobranie raportu
            U->>UI: Wybiera opcję pobrania raportu
            UI->>S: Żądanie generowania raportu
            S->>SS: Generuj raport z całego okresu
            SS->>DB: Pobierz pełne dane historyczne
            DB-->>SS: Zwraca dane historyczne
            SS->>SS: Kompiluje raport
            SS-->>S: Wygenerowany raport
            S->>S: Tworzy plik do pobrania (PDF/CSV)
            S-->>UI: Plik gotowy do pobrania
            UI-->>U: Rozpoczyna pobieranie pliku
            U->>U: Zapisuje raport lokalnie
        end
    end
