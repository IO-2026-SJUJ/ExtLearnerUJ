sequenceDiagram
    actor U as Użytkownik
    participant UI as Interfejs Testu
    participant S as System
    participant TS as Serwis Testów
    participant SS as Serwis Statystyk
    participant DB as Baza Danych

    U->>UI: Uruchamia test sprawdzający
    UI->>S: Żądanie rozpoczęcia testu
    S->>TS: Pobierz pytania testu dla materiału
    TS->>DB: Pobierz pytania powiązane z materiałem
    DB-->>TS: Zwraca zestaw pytań
    TS-->>S: Przygotowany test
    S-->>UI: Wyświetla pytania testu
    UI-->>U: Prezentuje pytania

    loop Dla każdego pytania
        UI->>U: Wyświetla pytanie
        U->>UI: Udziela odpowiedzi
    end

    U->>UI: Przesyła test (przycisk zakończ)
    UI->>S: Przesyła odpowiedzi
    S->>TS: Ocenianie testu
    TS->>DB: Pobierz klucz odpowiedzi
    DB-->>TS: Zwraca klucz
    TS->>TS: Ocenia odpowiedzi
    TS->>DB: Zapisz wynik testu
    DB-->>TS: Potwierdzenie zapisu
    TS-->>S: Wynik testu

    Note over S, SS: UC13 - Aktualizowanie statystyk

    S->>SS: Aktualizuj statystyki użytkownika
    SS->>DB: Pobierz obecne statystyki
    DB-->>SS: Zwraca statystyki
    SS->>SS: Aktualizuje poziom opanowania materiału
    SS->>DB: Zapisz zaktualizowane statystyki
    DB-->>SS: Potwierdzenie zapisu
    SS-->>S: Statystyki zaktualizowane

    S-->>UI: Prezentuje wynik testu
    UI-->>U: Wyświetla wynik i szczegóły odpowiedzi
