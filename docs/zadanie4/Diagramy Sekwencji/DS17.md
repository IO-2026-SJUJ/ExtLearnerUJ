sequenceDiagram
    actor U as Użytkownik
    participant UI as Edytor Testów
    participant S as System
    participant DB as Baza Danych

    U->>UI: Wybiera opcję tworzenia testu/ćwiczeń
    UI-->>U: Wyświetla edytor testów i ćwiczeń

    loop Definiowanie pytań/zadań
        U->>UI: Dodaje nowe pytanie/zadanie
        UI-->>U: Formularz pytania (treść, typ, odpowiedzi)
        U->>UI: Wypełnia treść pytania
        U->>UI: Definiuje odpowiedzi (poprawne/błędne)
        UI->>UI: Dodaje pytanie do listy
    end

    U->>UI: Zapisuje test/ćwiczenia
    UI->>S: Przesyła dane testu
    S->>S: Waliduje kompletność testu
    S->>DB: Zapisuje test i pytania
    DB-->>S: Potwierdzenie zapisu
    S->>DB: Dołącza test do materiału
    DB-->>S: Potwierdzenie powiązania
    S-->>UI: Test zapisany pomyślnie
    UI-->>U: Wyświetla potwierdzenie
