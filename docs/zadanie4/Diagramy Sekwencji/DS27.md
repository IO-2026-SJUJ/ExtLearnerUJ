sequenceDiagram
    actor U as Użytkownik
    participant UI as Interfejs Rankingu
    participant S as System
    participant RS as Serwis Rankingu
    participant DB as Baza Danych

    U->>UI: Otwiera ranking studentów
    UI->>S: Żądanie danych rankingowych
    S->>RS: Pobierz aktualny ranking
    RS->>DB: Pobierz wyniki i aktywność użytkowników
    DB-->>RS: Zwraca dane użytkowników
    RS->>RS: Sortuje użytkowników według kryterium (punkty, aktywność)
    RS->>RS: Przygotowuje listę rankingową
    RS-->>S: Lista rankingowa
    S-->>UI: Prezentuje ranking
    UI-->>U: Wyświetla aktualną listę rankingową

    opt Użytkownik zmienia kryterium sortowania
        U->>UI: Zmienia kryterium (np. wyniki testów, aktywność)
        UI->>S: Żądanie rankingu z nowym kryterium
        S->>RS: Sortuj według nowego kryterium
        RS->>RS: Przelicza ranking
        RS-->>S: Zaktualizowana lista
        S-->>UI: Prezentuje nowy ranking
        UI-->>U: Wyświetla zaktualizowany ranking
    end
