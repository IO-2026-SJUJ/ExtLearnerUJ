sequenceDiagram
    actor M as Moderator
    participant UI as Kolejka Prac
    participant S as System
    participant DB as Baza Danych

    M->>UI: Wybiera pracę z listy
    M->>UI: Klika opcję rezerwacji
    UI->>S: Żądanie rezerwacji pracy

    S->>DB: Sprawdź aktualny status pracy
    DB-->>S: Zwraca status pracy

    alt Praca już zarezerwowana
        S-->>UI: Komunikat - praca niedostępna
        UI-->>M: Wyświetla informację o niedostępności
    else Praca dostępna
        S->>DB: Przypisz pracę do moderatora (status: zarezerwowana)
        DB-->>S: Potwierdzenie przypisania
        S->>DB: Ustaw blokadę dla innych moderatorów
        DB-->>S: Potwierdzenie blokady
        S-->>UI: Praca zarezerwowana pomyślnie
        UI-->>M: Wyświetla potwierdzenie - praca przypisana
    end
