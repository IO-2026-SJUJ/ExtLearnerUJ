sequenceDiagram
    actor A as Administrator
    participant UI as Panel Zgłoszeń
    participant S as System
    participant DB as Baza Danych

    A->>UI: Otwiera panel zgłoszeń
    UI->>S: Żądanie listy zgłoszeń
    S->>DB: Pobierz zgłoszenia (z filtrami: status, typ, data)
    DB-->>S: Zwraca listę zgłoszeń
    S-->>UI: Prezentuje listę zgłoszeń
    UI-->>A: Wyświetla zgłoszenia z informacjami podstawowymi

    A->>UI: Wybiera zgłoszenie do rozpatrzenia
    UI->>S: Żądanie szczegółów zgłoszenia
    S->>DB: Pobierz szczegóły zgłoszenia
    DB-->>S: Zwraca dane zgłoszenia (typ, powód, autor, obiekt)
    S-->>UI: Prezentuje szczegóły
    UI-->>A: Wyświetla pełne dane zgłoszenia

    opt UC47 - Zgłoszenie wymaga dodatkowej analizy (extend)
        Note over A, DB: Przejście do UC47 - Rozpatrywanie zgłoszeń
        A->>UI: Rozpoczyna rozpatrywanie zgłoszenia
    end
